# Ajustes de stock

## ¿Qué es?
Una forma rápida de **sumar o restar** unidades a un producto (por inventario, mermas, diferencias físicas, etc.).

## ¿Para qué sirve?
Para que lo que ves en el sistema coincida con lo que tienes realmente.

## Cómo hacerlo
1. Menú **⋮** → **Ajustar stock**.
2. Revisa el **stock actual**.
3. Usa los botones **– / +** o escribe la cantidad final que **debería quedar**.
4. **Aplicar**.

## Consejos
- Úsalo después de un conteo real.
