# Alertas de stock

## Qué hace
Te avisa cuando un producto puede **quedarse corto**. En la tabla verás:
- 🔴 **Rojo**: sin stock.
- 🟡 **Amarillo**: bajo el objetivo.
- 🟢 **Verde**: todo ok.

Tip: En la barra superior puedes activar **“Mostrar umbral”** para ver `stock actual / objetivo`.

## Cómo configurarlo (por producto)
1. Menú **⋮** → **Configurar alertas**.
2. Elige un **modo**:
   - **A mano**: escribe el **mínimo** que quieres tener (ej. 20). Simple y directo.
   - **Según ventas (promedio)**: la app mira tus ventas recientes y propone un **consumo diario**. El objetivo será lo que vendes **en 1 día**.
   - **Según ventas para cubrir periodo**: igual que el anterior, pero además eliges **cuántos días** quieres cubrir (ej. 14). El objetivo será lo que vendes en esos días.
3. Revisa los valores y **Guarda**.

> ¿Cuál elijo?  
> - Si estás partiendo, usa **A mano**.  
> - Si ya tienes historial, **Según ventas** te ahorra cálculo y se adapta a tu realidad.
