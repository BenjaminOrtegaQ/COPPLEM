# Acciones masivas

## ¿Qué es?
Una barra que aparece cuando seleccionas varios productos para **aplicar cambios de una sola vez**.

## ¿Para qué sirve?
Para no ir producto por producto cuando quieras:
- **Establecer alertas** a muchos a la vez.
- **Eliminar** varios productos.

## Cómo usarla
1. Marca los cuadraditos de los productos que quieras.
2. Usa la barra que aparece:
   - **Establecer alertas**: Establece una alerta para todos los marcados.
   - **Eliminar seleccionados**: Borra todos los marcados.

## Consejo
- Úsalo después de una importación para asignar alertas (ej: cobertura 14 días a todos los productos recién importados).
- Marca el cuadradito de arriba para seleccionar todos los productos
