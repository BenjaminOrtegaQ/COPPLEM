# Ingresos totales (gráfico)

Muestra la evolución de ingresos en el período elegido:

- **Semanal**: últimos 7 días.
- **Mensual**: últimas semanas (vista por rangos Lun–Dom).
- **Anual**: últimos 12 meses.

## Cómo leerlo
- La **línea** marca la tendencia.
- Los **puntos** tienen etiquetas resumidas (4.8k, 1.2M).
- Debajo verás las **etiquetas de tiempo** (día/semana/mes).

**Consejos**
- Si la serie se ve plana, prueba cambiar el período a **Mensual** o **Anual**.
- Usa este bloque para detectar **picos** o **caídas** y cruzarlo con tu actividad.
