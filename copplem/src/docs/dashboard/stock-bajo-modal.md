# Lista de productos con stock bajo

Desde el KPI **Stock bajo**, abre un modal con el detalle:

- **Aro rojo**: sin stock.
- **Aro amarillo**: bajo el objetivo (mínimo o cobertura de días).
- Verás **nombre**, **SKU/ID**, **stock** y **categoría**.

## Qué hacer después
- Clic en **“Ir a productos”** para:
  - Ajustar stock.
  - Cambiar alertas.
  - Revisar precios y compras.
