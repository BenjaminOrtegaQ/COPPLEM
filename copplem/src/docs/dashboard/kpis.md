# Indicadores rápidos (arriba)

Los recuadros superiores muestran:

- **Ventas hoy**: cantidad de transacciones del día.
- **Ingresos diarios / semanales / mensuales**: monto total vendido.
- **Stock bajo**: cuántos productos están en alerta.

## Ver productos en alerta
1. Haz clic en el icono 🛈 del recuadro **Stock bajo**.
2. Se abrirá una lista con los productos en **rojo** (sin stock) o **amarillo** (bajo objetivo).
3. Usa **“Ir a productos”** para gestionar stock/alertas.

> Tip: configura umbrales en *Productos → Configurar alertas* para que este número sea más útil.
