# COPPLEM
Proyecto de Tesis | Sistema de gestión de negocios para microemprendedores - Jorge Cartez y Benjamín Ortega

COPPLEM es una herramienta diseñada para apoyar a los pequeños negocios a través de la gestión de su inventario, registrar ventas y visualizar el estado del negocio con métricas de interés. Se desarrolla principalmente por la necesidad de las personas de tener un sistema gratuito y offline al que puedan acceder para dar sus primeros pasos en el mundo del emprendimiento.

El proyecto empieza por una hipótesis generada por experiencias de mi compañero y yo, donde vemos una oportunidad de mejora para este nicho. Para dar mayor robustez y confirmar la problemática, se investiga al respecto.

Con la investigación (Encuestas y búsquedas en internet) se reúnen requerimientos que son clave a la hora de desarrollar el proyecto. La información permite determinar que el sistema debe ser intuitivo, offline y gratuito como principales focos a la hora de desarrollar. Dentro de las funciones más importantes del proyecto podemos encontrar:

- Gestión de productos (CRUD).
- Registro de ventas e historial con cálculo automático.
- Inventario con alertas de stock.
- Dashboard con métricas de interés.
- Reportes y exportación a Excel.
- Predicción simple de Stock y recomendaciones de reposición basadas en historial del negocio.
- Recomendaciones preestablecidas para orientar al usuario.
  
Dichas recomendaciones son dadas por el asistente Copply, un robot amigable cuya funcionalidad es apoyar al usuario y crear un ambiente más grato mientras gestiona su negocio.
